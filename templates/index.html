🔌 WordPress Gravity Forms Integration (Free Access / Lead Gen)

Automate giving free access (e.g., for trials or lead magnets) directly from your WordPress site. This webhook automatically processes a 100% free order and emails the Telegram link instantly.

Step-by-Step Setup Guide:

  1. Install and activate the Webhooks Add-On in Gravity Forms.
  2. In your form, add fields for Name, Email, and Phone.
  3. Add a Hidden Field. Set its default value to your target Plan ID (e.g., PLN12345).
  4. Go to Settings > Webhooks and create a new feed.
  5. Set Request URL to the Webhook Target URL above.
  6. Set Request Method to POST and Format to JSON.
  7. Under Request Headers, add Name: X-API-Key and Value: (Your Secret API Key).
  8. Under Request Body, choose Select Fields and map the keys exactly as: name, email, mobile, plan_id.
  9. Crucial: In your TG Manager Dashboard, edit the specific Plan/Link and change "External Webhook Access" to Allow Free Webhook Access.
